# PC에서 ChatGPT로 이미지 최대 20장 생성하기

이 방식은 OpenAI API를 사용하지 않습니다. PC Edge 브라우저에서 ChatGPT 웹을 열고, 프로그램이 프롬프트를 입력합니다.

## 준비

1. PC에 Node.js LTS를 설치합니다.
2. 이 프로젝트 폴더에서 `run-pc-automation.bat`를 실행합니다.
3. 브라우저가 열리면 `ChatGPT 로그인/열기`를 누릅니다.
4. 열린 ChatGPT 브라우저에서 로그인합니다.

첫 실행 때는 Playwright Chromium이 설치되어 시간이 조금 걸릴 수 있습니다.

## 사용 흐름

```text
PC 프로그램 실행
↓
프롬프트 최대 20개 붙여넣기
↓
프롬프트 1~20 자동 분리
↓
프로그램 실행
↓
PC 브라우저 ChatGPT에서 이미지 생성
↓
PC 저장 폴더에 모음
↓
Google Drive / OneDrive / 카카오톡 / USB 등으로 폰에 전달
↓
폰에서 숏츠 제작
```

## 모드

### 전체 이미지를 한 번에 요청

기본값입니다. 프로그램이 최대 20개 프롬프트를 하나의 요청으로 합쳐 ChatGPT에 전송합니다.

장점:
- 전송 한 번
- 사용량이 단순함
- 폰에서 하던 수동 작업보다 훨씬 빠름

주의:
- ChatGPT가 반드시 요청 개수만큼 모두 생성한다는 보장은 없습니다.
- 한 장에 여러 컷으로 합치지 말라고 프롬프트에 포함해두었습니다.

### 순서대로 1개씩 요청

체크를 끄면 프롬프트를 순서대로 하나씩 전송합니다.

장점:
- 각 이미지가 독립적으로 생성될 가능성이 높음

주의:
- 시간이 더 오래 걸림
- ChatGPT 사용량 제한에 더 빨리 걸릴 수 있음

## 저장 위치

프로그램 폴더 아래에 저장됩니다.

```text
ShortsAutoOutput/
  프로젝트명/
    prompts.txt
    log.txt
    다운로드된 이미지 파일
```

ChatGPT UI가 다운로드 버튼을 직접 노출하지 않는 경우 자동 다운로드가 실패할 수 있습니다. 그때는 열린 ChatGPT 브라우저에서 이미지를 직접 저장하세요.

## 폰으로 옮기는 방법

가장 쉬운 방법:

- Google Drive 폴더에 업로드
- OneDrive 폴더에 업로드
- 카카오톡 나에게 보내기
- 네이버 MYBOX

프로젝트 폴더의 이미지 파일을 폰에서 내려받아 숏츠 제작 앱에 넣으면 됩니다.
