# PC에서 ChatGPT로 이미지 최대 20장 생성하기

이 방식은 OpenAI API를 사용하지 않습니다. PC의 Microsoft Edge에서 ChatGPT를 열고, 로컬 프로그램이 프롬프트를 입력합니다.

## 실행

1. `run-pc-automation.bat` 실행
2. 브라우저에서 `http://localhost:3737` 열림
3. ChatGPT가 로그인되어 있는지 확인
4. 프롬프트를 붙여넣고 `프롬프트 1~20 자동 분리`
5. `ChatGPT 연결 확인`
6. `프로그램 실행`

## 기존 Edge ChatGPT 탭에 연결하기

이미 열어둔 ChatGPT 마지막 창에서 바로 작업하려면 Edge가 원격 디버깅 모드로 실행되어 있어야 합니다.

가장 쉬운 방법:

1. 열려 있는 Edge 창을 모두 닫기
2. `run-edge-debug-chatgpt.bat` 실행
3. 열린 Edge에서 ChatGPT 로그인
4. 원하는 GPT/프로젝트/채팅 화면을 열어두기
5. `run-pc-automation.bat` 실행
6. 화면에서 `ChatGPT 연결 확인` 클릭
7. 연결되었다고 나오면 `프로그램 실행`

일반 방식으로 이미 켜져 있던 Edge 창에는 브라우저 보안상 자동으로 붙을 수 없습니다. 이 경우 프로그램이 별도 자동화용 Edge를 열 수 있습니다.

## 모드

### 전체 이미지를 한 번에 요청

기본값입니다. 최대 20개 프롬프트를 하나의 요청으로 묶어 ChatGPT에 전송합니다.

장점:
- 전송 횟수가 적음
- 사용 방식이 단순함

주의:
- ChatGPT가 요청한 개수만큼 모두 생성한다고 보장할 수는 없습니다.
- 한 장에 여러 컷을 합치지 말라는 문구를 자동으로 포함합니다.

### 순서대로 1개씩 요청

체크를 끄면 프롬프트를 순서대로 하나씩 전송합니다.

장점:
- 각 이미지가 독립적으로 생성될 가능성이 높음

주의:
- 시간이 더 오래 걸림
- ChatGPT 사용량 제한에 빨리 걸릴 수 있음

## 저장 위치

프로그램 폴더 아래에 저장됩니다.

```text
ShortsAutoOutput/
  프로젝트명/
    prompts.txt
    log.txt
    다운로드된 이미지 파일
```

현재 PC 자동화는 이미지 다운로드를 진행하지 않습니다. 프롬프트를 ChatGPT에 입력하고 이미지 생성을 완료하는 것까지만 자동으로 처리합니다. 생성된 이미지는 열린 Edge ChatGPT 화면에서 직접 확인하거나 저장하세요.

## 한계
